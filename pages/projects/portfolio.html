Personal Portfolio Website

This website is currently deployed and hosted on Netlify!


What my page looked like when I started.


Initially starting this website I had prior experiences with HTML, CSS and Javascript. However I didn't really get to use those tools as much during my time at UW so alot of the things I learned have become rusty. Also I never really got to learn the basics of those programs. So I decided what better way to learn web development than building a personal portfolio.

While working on this website I got to go over alot of the fundamentals and basics of web development such as styling pages with css instead of copying and pasting the same information on html files. I have also learned alot about positioning elements with flex boxes, margins, etc. While learning how to build this website I also got to try many other different tools. Some of those tools include NodeJS and Gatsby. I also tried out bootstrap which was really convenient to use but I wanted to personalize my page more and have more control over the styling. Doing so also helps me learn a lot of different techniques.

At the moment I am still working on fixing the structure of some parts of the UI and plan to add more features as I develop this space more. There is a lot more work to be done on this website!
